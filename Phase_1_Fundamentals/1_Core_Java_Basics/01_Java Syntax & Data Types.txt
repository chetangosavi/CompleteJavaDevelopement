Basic Syntax:

    + Syntax :

        public class Main {
            public static void main(String[] args) {
                System.out.println("Hello, Chetan!");
            }
        }

        Explanation:
            public class Main: Every Java program must have at least one class.
            public static void main(String[] args): The entry point of any Java program.
            System.out.println(): Used to print output.

    
    + Java Data Types :

        Java has two categories:
            Primitive Data Types (8 total)
            Non-Primitive Data Types (like String, Arrays, Classes)
        
        Primitive Data Types

            Data Type	                     Size                   Example
            byte	                        1 byte	                byte a = 10;
            short	                        2 bytes	                short b = 1000;
            int	                            4 bytes	                int c = 100000;
            long	                        8 bytes	                long d = 999999L;
            float	                        4 bytes	                float e = 5.75f;
            double	                        8 bytes	                double f = 19.99;
            char	                        2 bytes	                char g = 'A';
            boolean	                        1 bit	                boolean h = true;


         Note: Always add f or L for float and long values.
